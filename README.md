# 🤖 AI Article Builder

An AI-powered article generation web application built using Flask, Python, Ollama, MySQL, HTML, CSS, and JavaScript.

## 🚀 Features

- 🔐 User Authentication (Login & Registration)
- ✍️ AI-Powered Article Generation
- 📈 SEO-Friendly Content Creation
- 📚 Article History Management
- 📄 PDF Export
- 🌙 Dark Mode Interface
- 📱 Responsive Design
- 🤖 Ollama (Qwen2.5:3B) Integration

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Python
- Flask

### Database
- MySQL

### AI Model
- Ollama
- Qwen2.5:3B
